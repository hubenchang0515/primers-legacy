# HTML 的 <h5> 标签

请查看 HTML 元素帮助手册 了解更多 HTML 元素。

HTML <h1>–<h6> 标题 (Heading) 元素呈现了六个不同的级别的标题，<h1> 级别最高，而 <h6> 级别最低。

  • 通常情况下一个页面应当有且只有一个 <h1> 标题

  • 标题级别应当逐级使用，不应跳级

  • 不要为了生成 粗体 或 大号 的文本而使用标题

# 示例

<h1> 文章标题 </h1>

<h2> 第一章：引言 </h2>

<p>
    这是一个段落示例。在这里可以写下文章的引言部分，介绍文章的主题和主要内容。通常引言部分会简要概括全文，吸引读者的注意力。
</p>

<h2> 第二章：主要内容  </h2>

<p>
    这是第二个段落。在这里可以展开详细说明文章的主要观点或论据。使用段落(p)元素可以将内容分成易于阅读的小块。
</p>

<p>
    这是第三个段落。当需要表达新的观点或转换话题时，可以开始新的段落。这样能使文章结构更清晰，读者更容易跟随你的思路。
</p>

<h3> 2.1 子章节示例  </h3>

这是子章节下的段落。使用h3元素可以创建更深层次的章节结构，适用于更详细的内容划分。

<h2> 第三章：结论  </h2>

<p>
    最后一个段落通常用于总结全文或提出结论。在这里可以重申主要观点，或者提出进一步的思考问题。
</p>

<p>
    这是文章的结尾段落。感谢读者的阅读，也可以在这里添加号召性语句或后续行动的提示。
</p>
