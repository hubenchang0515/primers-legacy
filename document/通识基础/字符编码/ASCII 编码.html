# ASCII 编码表

ASCII（American Standard Code for Information Interchange，美国信息交换标准代码）是一种字符编码标准，用于表示一组特定的 95 个（英语）可打印字符和 33 个控制字符——共 128 个代码点，使用一个字节的低 7 位表示。这组可用的标点符号对计算机语言和文本标记的语法产生了重大影响。ASCII 极大地影响了现代计算机使用的字符集的设计，后续几乎所有编码方案的前 128 个编码都与 ASCII 兼容。

参考：https://en.wikipedia.org/wiki/ASCII

# 高位 ASCII 码

高位 ASCII 码即字节最高位为 1 的特殊字符：

  • ASCII 码是 7 位，范围是 0–127

  • 高位 ASCII 范围是 128-255

在 UNIX 上通常使用 M- 前缀进行显示。例如字节编码为 233，它的低 7 位为 105，对应 ASCII 字符为 i，因此显示为 M-i。
