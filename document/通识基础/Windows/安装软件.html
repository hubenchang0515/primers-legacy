# 安装软件

Windows 的软件安装方式主要使用安装包，以 VSCode 为例，从 Visual Studio Code 的官方网站 下载 VS Code 的安装包：

浏览器下载历史中，点击即可打开下载目录或直接运行安装包：

通常浏览器下载的文件默认保存在 C：\Users\${USER_NAME}$\Downloads\ 路径中：

双击运行安装包后开始安装，大多数软件通常会让用户阅读许可协议，如果拒绝则无法安装，因此只能选择 同意，然后点击 下一步：

接下来通常是安装过程中最重要的一步，选择安装路径，如果不清楚该怎么选，就保持默认选项点击 下一步 即可：

然后通常会询问是否添加到开始菜单（即 Window LOGO 按键打开的菜单），如果不清楚该怎么选，就保持默认选项点击 下一步 即可：

再然后会有一些安装选项让用户选择，对于大多数软件，如果不清楚该怎么选，就保持默认选项点击 下一步 即可：

对于 VS Code，建议勾选全部选项

最后会让你确认前面的所有操作，确认无误后点击 安装 ：

等待安装完成后点击 完成 即可，因为前面勾选了 创建桌面快捷方式，因此安装完毕后能够在桌面上看见 VS Code 的快捷方式。

# 卸载软件

通过安装包安装的软件通常会附带卸载程序，能够快速进行卸载，它通常在软件安装的目录中。

右击快捷方式，点击 属性 查看快捷方式的属性，可以看见对于可执行文件的路径：

点击 打开文件所在位置 即可进入安装目录：

卸载程序的文件名通常叫做 uninstall 之类的，这里的 unins000.exe 就是卸载程序，双击即可进行卸载。
