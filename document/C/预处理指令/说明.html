# C 语言的预处理指令

C 语言的预处理指令是在编译器的预处理阶段对代价进行加工的特殊的特殊命令。

预处理命令全部以井号（#）开头，并且由于不是 C 语言本身的语句，因此不以分号（；）结尾。

目前包含以下指令：

名称 指令
文件包含指令 #include
宏定义指令 #define #undef
条件编译指令 #if #elif #else #endif #ifdef #ifndef #elifdef #elifndef
行号修改指令 #line
错误和警告指令 #error #warning
文件嵌入指令 #embed
编译器指令 #pragma

其中 #elifdef，#elifndef，#warning 和 #embed 是 C23 标准新增的指令，低版本编译器可能无法编译。

使用 gcc 的 -E 选项可以查看预处理的结果。例如：

#include <stdio.h>

int main(void)
{
    printf("hello world\n");
    return 0;
}

执行 gcc -E -P main.c 得到：

  • -E 表示只执行预处理

  • -P 表示不显示行号等额外信息
