# C 语言的预处理文件包含指令 #include

文件包含指令（#include）使用被包含的文件内容替换自身。有以下两种形式：

#include </path/to/file>    // 相对路径的起点为编译器的头文件搜索路径

#include "/path/to/file"    // 相对路径的起点为当前文件所在目录

  • 使用尖括号（</path/to/file>）包含文件时，如果是相对路径，则以编译器的头文件搜索路径（例如 /usr/include）为起点

  • 使用双引号（"/path/to/file"）包含文件时，如果是相对路径，则以当前源文件所在目录为起点

例如，有如下两个文件：

CCCCCCCC

AAAAAAAA
#include "header.h"
BBBBBBBB

则 main.c 经过预处理后得到：

AAAAAAAA
CCCCCCCC
BBBBBBBB

可以看到搜索路径包含：

  • /usr/lib/gcc/x86_64-linux-gnu/13/include

  • /usr/local/include

  • /usr/include/x86_64-linux-gnu

  • /usr/include

可以编辑环境变量 C_INCLUDE_PATH 添加头文件搜索路径，多个路径使用冒号（:，Linux）或分号(;，Windows) 分隔。例如：

$ export C_INCLUDE_PATH="/opt/lib1/include:/opt/lib2/include"
$ gcc -v main.c
