# Lua 的 repeat 循环语句

Lua 的 repeat 循环语句语法为：

repeat
    代码块
until 条件

  • 循环执行代码块，直到条件为真

graph TD
    START([开始]) --> 循环体 --> COND{循环条件?} -- 假 --> 循环体
    COND -- 真 --> END([结束])

示例：

local i = 0
repeat
    print("当前 i 的值为", i)
until i <= 0    -- 直到 i 小于或等于 0

运行结果：

当前 i 的值为   0

  • 即使一开始 i 的值就满足结束条件，循环也会执行一次
